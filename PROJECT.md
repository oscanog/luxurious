# Luxurious

**Elite org dashboard. Convex + React.**

Real-time sync. Premium UI. High-end management.

## Vision

User experience first. Management effortless. Visuals stunning.

## Features

- **Org Chart**: Interactive hierarchy. React Flow.
- **Members**: Team management. Roles.
- **Invitations**: Flow + tracking.
- **Auth**: Convex Auth. Secure.
- **Design**: Dark mode. Smooth transitions.

## Tech

- **Backend**: Convex (DB, Func, Auth)
- **Frontend**: React 19, Vite 8
- **Styling**: Tailwind CSS 4
- **Flow**: @xyflow/react
- **Icons**: Lucide
