# Feature: Learn to Trade
Simulate market. Learn risk. Master Luxurious trading.

## Vision
Game-ify trading. User learn without lose money. Transition to real trade later.

## Page Map
1. **Market Hub**: Live-style price list. Trending asset.
2. **Sim Canvas**: Pro chart (TradingView style). Indicators.
3. **Trade Terminal**: Quick buy/sell. Leverage slider. Margin info.
4. **Active Orders**: Current sim position. Real-time P/L.
5. **Academy**: Module-based learning. Quiz. Rewards.

## UI/UX Spec (Senior)
- **Vibe**: Elite terminal. Glassmorphism. High-contrast price action.
- **Color**: Bull (Green: `hsl(152 69% 42%)`), Bear (Red: `hsl(0 84% 61%)`).
- **Interaction**: One-click order. Instant feedback.
- **Charts**: Dark theme. Candle-stick pattern.
