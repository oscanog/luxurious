# Knowledge
Patterns + rules.

## Convex + React
- **Schema First**: Define types in `convex/schema.ts`. Type safety win.
- **Component Auth**: Use `Authenticated`/`Unauthenticated` helpers.
- **Action for Effects**: Third-party API? Use `action`.

## UI/UX
- **Dark Mode**: Deep palette only. Background dark.
- **Icons**: Lucide always.
- **Vite Alias**: `@/` for `src/`.

## GSD + Caveman
- **Slice**: 3-5 tasks. Keep context lean.
- **Must-Haves**: Verify outcomes mechanically (file, lint, output).
- **Caveman Mode**: Concise word. Save token. Brain big.
