# Requirements
Capability contract.

## 1. Auth & Security
- [x] Login/logout (Convex Auth).
- [x] Protected routes (Auth only).
- [ ] Role-based access (RBAC).

## 2. Org Chart
- [x] Visualize hierarchy.
- [ ] Add/Remove nodes on chart.
- [ ] Edit details in nodes.
- [ ] Drag-drop reorganize.

## 3. Member Management
- [x] List members.
- [ ] Search + Filter.
- [ ] Update roles/perms.
- [ ] Remove members.

## 4. Invitations
- [x] List pending/accepted.
- [ ] Send via email.
- [ ] Resend/Revoke.
- [ ] Batch support.

## 5. UI/UX
- [x] Responsive layout.
- [x] Dark mode.
- [x] Real-time updates.
- [ ] Feedback on action.

## Out of Scope
- Mobile native.
- Public landing page.
- Financial integrations.
